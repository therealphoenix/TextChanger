ТРЕБОВАНИЯ.

1. При выполнении задания создать и обязательно использовать классы
Word и Sentence.

2. Приложение должно быть написано на любом из языков: С++, С# или Java.
3. При кодировании должны быть использованы соглашения об оформлении
кода для соответствующего языка – code convention.

4. По возможности использовать принципы ООП: наследование,
полиморфизм, инкапсуляция.

5. Не использовать графический интерфейс! Приложение должно быть
консольным. но консольное меню должно быть минимальным

6. На собеседование приносить на флеш-носителе исходный код и
откомпилированное приложение.
